Fisch is a Roblox experience originally developed by Nate (@WoozyNate). Its official release was on October 5, 2024. It is an open-world adventure and exploration game in which the primary goal is to catch Fish for profit, bestiary progression and completing quests.

Version 1.59.0
Version 1.59 is the sixtieth release version of Fisch, which introduced the Mineshaft, as well as reworked Merlin. In the latest Fisch changes in 1.59.1, they introduced the new limited Fischgiving 2025 event.
